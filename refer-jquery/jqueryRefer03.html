종류 jQuery Javascript 설명
Attributes .attr() setAttribute()
getAttribute()		 선택한 요소의 속성 값을 설정합니다.
.removeAttr() - 선택한 HTML 요소에서 하나 이상의 속성을 제거합니다.
.prop() - 선택한 JavaScript 요소의 속성 값을 설정하거나 반환합니다.
.removeProp() - 선택한 JavaScript 요소에서 하나 이상의 속성을 제거합니다.
.val() - 선택한 폼 요소의 속성 값을 설정하거나 반환합니다.
CSS .css() - 선택한 요소에 CSS 속성 값을 설정합니다.
.addClass() .classList.add() 클래스를 추가합니다.
.removeClass() .classList.remove() 클래스를 삭제합니다.
.toggleClass() .classList.toggle() 클래스를 추가/삭제합니다.
.hasClass() .classList.contains() 클래스를 검색합니다.
Dimensions .width() - 요소의 가로 값(패딩/보더/마진 불포함) 을 가져오거나 설정합니다.
.innerWidth() element.clientWidth 요소의 가로 값(패딩 포함) 을 가져오거나 설정합니다.
.outerWidth() element.offsetWidth 요소의 가로 값(패딩/보더 포함) 을 가져오거나 설정합니다.
.outerWidth(true) - 요소의 가로 값(패딩/보더/마진 포함) 을 가져오거나 설정합니다.
.height() - 요소의 세로 값(패딩/보더/마진 불포함) 을 가져오거나 설정합니다.
.innerHeight() element.clientHeight 요소의 세로(패딩 포함) 값을 가져오거나 설정합니다.
.outerHeight() element.offsetHeight 요소의 세로 값(패딩/보더 포함) 을 가져오거나 설정합니다.
.outerHeight(true) - 요소의 세로 값(패딩/보더/마진 포함) 을 가져오거나 설정합니다.
$(document).width() - 문서의 가로 값을 가져오거나 설정합니다.
$(window).width() window.innerWidth 브라우저의 가로 값을 가져오거나 설정합니다.
$(window).width() window.innerHeight 브라우저의 세로 값을 가져오거나 설정합니다.
offset .offset().top / .offset().left .offsetTop / .offsetLeft 요소의 좌표 값(문서를 기준)을 가져오거나 설정합니다.
.offsetParent() - 요소의 좌표 값(부모 위치를 기준)을 가져오거나 설정합니다.
.position() - 요소의 좌표 값(기준점 기준)을 가져오거나 설정합니다.
.scrollLeft() - 요소의 가로 스크롤 값을 가져오거나 설정합니다.
.scrollTop() - 요소의 세로 스크롤 값을 가져오거나 설정합니다.
Data .data() - 선택한 요소의 데이터를 설정하거나 반환합니다.
.removeData() - 선택한 요소의 데이터를 삭제합니다.

01. CSS - addClass() / removeClass()

클릭하면 모든 이미지에 클래스(active)를 추가합니다. 클릭하면 모든 이미지에 클래스(active)를 삭제합니다. 
{
    $(".btn1-1").click(function(){
        $(".ex1 div").addClass("active");
    });
    $(".btn1-2").click(function(){
        $(".ex1 div").removeClass("active");
    });
}

02. CSS - toggleClass()

이미지를 클릭하면 이미지에 클래스(active)를 추가/삭제합니다. 
{
    //모든 이미지 선택
    $(".ex2 div").click(function(){
        $(".ex2 div").toggleClass("active");
    });
    //클릭한 이미지만 선택
    $(".ex2 div").click(function(){
        $(this).toggleClass("active");
    });
}

03. CSS - hasClass()

리셋 클릭하면 active 클래스를 찾은 후에 클래스를 제거합니다. 
{
    $(".btn3-0").click(function(){
        $(".ex3 > div:nth-child(1)").addClass("active");
        $(".ex3 > div:nth-child(4)").addClass("active");
    });
    $(".btn3-1").click(function(){
        if( $(".ex3 > div").hasClass("active") ) {
            $(".ex3 > div.active").removeClass("active");
        }
    });
}

여기는 텍스트 영역입니다.

클릭하면 원의 너비(width) 값을 구합니다. 클릭하면 원의 너비(innerWidth) 값을 구합니다. 클릭하면 원의 너비(outerWidth) 값을 구합니다. 클릭하면 원의 너비(outerWidth(true)) 값을 구합니다. 클릭하면 원의 높이(htight) 값을 구합니다. 클릭하면 원의 높이(innerHeight) 값을 구합니다. 클릭하면 원의 높이(outerHeight) 값을 구합니다. 클릭하면 원의 높이(outerHeight(true)) 값을 구합니다.
클릭하면 샘플 박스의 너비/높이 값을 구합니다. 클릭하면 문서의 너비/높이 값을 구합니다. 클릭하면 브라우저의 너비/높이 값을 구합니다.

    $(".text4").text("이 원의 " + name + "값은 " + size + "px 입니다.");

05. Dimensions - width()/ height()

리셋 클릭하면 원의 너비 값을 200px 변경합니다. 클릭하면 원의 높이 값을 200px 변경합니다. 클릭하면 원의 너비, 높이 값을 130px 변경합니다. 
{
    $(".btn5-0").click(function(){
        $(".circle5").width("70").height("70")
    });
    $(".btn5-1").click(function(){
        $(".circle5").width("200")
    });
    $(".btn5-2").click(function(){
        $(".circle5").height("200")
    });
    $(".btn5-3").click(function(){
        $(".circle5").width("130").height("130")
    });
}

06. offset(), position()

여기는 텍스트 영역입니다.

클릭하면 원의 x축 위치 값(offset)을 구합니다. 클릭하면 원의 y축 위치 값(offset)을 구합니다. 클릭하면 원의 x축 위치 값(position)을 구합니다. 클릭하면 원의 y축 위치 값(position)을 구합니다. 
{
    $(".btn6-1").click(function(){
        $(".text6").text("이 원의 x축 값은" + $(".circle6").offset().left + "px 입니다.");
    });   
    $(".btn6-2").click(function(){
        $(".text6").text("이 원의 y축 값은" + $(".circle6").offset().top + "px 입니다.");
    });   
    $(".btn6-3").click(function(){
        $(".text6").text("이 원의 x축 값은" + $(".circle6").position().left + "px 입니다.");
    });   
    $(".btn6-4").click(function(){
        $(".text6").text("이 원의 y축 값은" + $(".circle6").position().top + "px 입니다.");
    });   
}

07. offset(), position()

offset().left = 0

offset().top = 0

position().left = 0

position().top = 0

원을 드래그하세요! 
{
    $( ".circle7" ).draggable({
        start: function() {
            $(".text7-5").text("드래그가 시작되었습니다.");
        },
        drag: function() {
            $(".text7-1 span").text( parseInt($(".circle7").offset().left) + "px");
            $(".text7-2 span").text( parseInt($(".circle7").offset().top) + "px");
            $(".text7-3 span").text( parseInt($(".circle7").position().left) + "px");
            $(".text7-4 span").text( parseInt($(".circle7").position().top) + "px");
        },
        stop: function() {
            $(".text7-5").text("드래그가 끝났습니다.");
        }
    });
}
