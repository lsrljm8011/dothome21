종류 제이쿼리 예시 자바스크립트 설명
필터링 .eq() ⭐ $("selector").eq(3) 선택한 요소의 인덱스 번호에 해당하는 요소를 찾습니다.
.filter() $("selector").filter() 선택한 요소에서 선택자, 변수, 함수를 통해 선택하는 메서드입니다.
.first() $("selector").first() 선택된 요소에서 첫 번째 요소를 찾습니다.
.has() ⭐ $("selector").has() 선택한 요소에서 주어진 선택자가 있으면 선택한 요소를 선택합니다.
.is() $("selector").is() 선택한 요소에서 주어진 선택자가 있는지 판별합니다.
.last() $("selector").last() 선택된 요소에서 마지막 번째 요소를 찾습니다.
.map() $("selector").map() 선택된 요소에서 함수를 사용하여 새로운 집합을 만듭니다.
.not() $("selector").not() 선택된 요소에서 조건에 맞지(반대) 않는 것을 선택합니다.
.slice() $("selector").slice() 선택된 요소에서 조건의 범위로 재선택합니다.
트리구조 .children() $("selector").children() 선택한 요소의 바로 아래 자식 요소만 선택합니다.
.closets() $("selector").closets() 선택한 요소를 포함하면서 가장 가까운 상위 요소를 선택합니다.
.find() ⭐ $("selector").find() 선택한 요소의 자식 요소 중 조건에 맞는 요소를 선택합니다.
.next() ⭐ $("selector").next() 선택한 요소의 다음 요소를 선택합니다.
.nextAll() $("selector").nextAll() 선택한 요소의 다음 모든 형제 요소를 선택합니다.
.nextUntil() $("selector").nextUntil() 선택한 요소의 다음에 위치한 요소를 조건에 맞을 때까지 찾습니다.
.parent() $("selector").parent() 선택한 요소의 부모 요소를 선택합니다.
.parents() ⭐ $("selector").parents() parentNode 선택한 요소의 모든 부모 요소를 선택합니다.
.parentUntil() $("selector").parentUntil() 선택한 요소의 부모 요소 중 조건에 맞을 때까지 찾습니다.
.prev() ⭐ $("selector").prev() 선택한 요소의 이전 요소를 선택합니다.
.prevAll() $("selector").prevAll() 선택한 요소의 부모 요소 중 조건에 맞을 때까지 찾습니다.
.prevUntil() $("selector").prevUntil() 선택한 요소의 부모 요소 중 조건에 맞을 때까지 찾습니다.
.siblings() $("selector").siblings() 선택한 요소의 형제 요소를 모두 찾습니다.
기타 .add() $("selector").add() 선택된 요소와 새로운 요소에 추가로 선택합니다.
.addBack() $("selector").addBack() 선택된 요소와 이전 요소를 선택합니다.
.contents() $("selector").contents() 선택한 요소와 일치하는 콘텐츠 요소를 선택합니다.
.each() ⭐ $("selector").each() forEach 선택한 여러 요소들을 순차적으로 실행합니다.
.end() $("selector").end() 선택한 요소가 선택되기 이전의 요소를 선택합니다.

  • $(".class")
  • $("#id")

  • $("*")
